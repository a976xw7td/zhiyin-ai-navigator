Zhiyin AI Navigator - Chrome Extension Install Guide

1. Unzip this package.
2. Open Chrome and visit chrome://extensions/
3. Turn on Developer mode.
4. Click Load unpacked.
5. Select the unzipped folder: ZhiyinAI_Navigator_Install
6. Open the extension from Chrome toolbar.
7. Fill API Keys in Settings if AI / voice features are needed.

Do not remove manifest.json, background, content, lib, offscreen, sidepanel, assets.
