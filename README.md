# 智引 AI 导航导师 (Zhiyin AI Navigator)

> 帮助高校师生在在线教学平台上通过语音快速定位功能、理解课程内容、完成学习任务的 Chrome 浏览器扩展。

## 功能

- **语音导航**：通过语音指令快速定位在线教学平台的各项功能
- **智能问答**：理解课程内容，回答学习相关问题
- **内容摘要**：自动提取和总结页面关键信息
- **PDF 导出**：将对话和课程内容导出为 PDF

## 安装

1. 下载本仓库（Code → Download ZIP 或 `git clone`）
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择本仓库根目录
6. 安装完成，快捷键 `Command+Shift+Y`（Mac）或 `Ctrl+Shift+Y`（Windows）打开

## 技术栈

Chrome Extension Manifest V3 · Side Panel API · TTS · DeepSeek API · SiliconFlow API
